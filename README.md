## Coco/R for C#

Coco/R is a parser generator written by

* Hanspeter Mössenböck
* Markus Löberbauer
* Albrecht Wöß

from the University of Linz

and

* Pat Terry

from Rhodes University.

## Building

```
$ xbuild /t:build Coco.csproj
```

## Usage

```
$ ./Coco.exe your_file.atg
```
